# 🥑 3주차 실습 결과 보고서: 체적 및 표면적 수치 적분

## 1. 실습 개요
본 실습에서는 OpenCV를 활용하여 아보카도의 프로파일을 추출하고, 큐빅 스플라인 보간법과 수치 적분(Simpson, Trapezoidal) 공식을 사용하여 아보카도의 체적을 계산하였습니다.

## 2. 주요 실습 파일 및 변동 사항
- `avocado_profile.py`: 이미지 기반 윤곽선 검출 및 프로파일 추출 모듈 최적화
- `step1_interpolation.py`: 큐빅 스플라인 보간 시각화 코드 실행
- `step2_volume.py`: 수치 적분 알고리즘 구현 및 정밀도 비교 분석
- `step3_3d_visualization.py`: 3D 회전체 시각화 결과 확인

## 3. 수치 적분 결과 분석 (n=100 기준)
| 항목 | 계산 결과 (cm³) | 교재 대비 오차 (%) |
| :--- | :---: | :---: |

| **교재 수동 계산값** | 334.1 | - |

## 4. 고찰 및 결론
- **수렴성 확인**: 분할 수(n)가 증가할수록 Simpson 공식과 사다리꼴 공식의 차이가 줄어들며 안정적인 체적값으로 수렴함을 확인하였습니다. (n=1000에서 차이 0.000000)
- **정밀도**: Simpson 공식이 2차 포물선 근사를 사용하므로, 동일 분할 수에서 사다리꼴 공식(직선 근사)보다 이론적으로 더 높은 정밀도를 제공함을 확인하였습니다.
- **이미지 기반 추출**: 자동 검출된 프로파일이 교재의 수동 측정 데이터와 약 1.4%의 미세한 차이를 보였으며, 이는 실제 시료의 비대칭성 및 이미지 전처리 과정에서의 미세한 오차가 반영된 결과로 판단됩니다.
